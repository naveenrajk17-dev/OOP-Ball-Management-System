BallManagementSystem is a menu-driven console application where a Bag can store different types of Ball objects such as:

TennisBall
BasketBall

The application allows the user to:
Add a ball
Remove a ball
Search for a ball
Display all balls
Play a game simulation
This project is designed to demonstrate practical implementation of OOP concepts in Java.
